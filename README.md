# 📄 Спецификация слайдера проектов

## 📦 Структура

| Файл/Папка   | Назначение                                              |
| ------------ | ------------------------------------------------------- |
| `index.html` | Разметка слайдера                                       |
| `style.css`  | Стили (фиксированная вёрстка)                           |
| `script.js`  | Логика переключения и синхронизации                     |
| `img/`       | Изображения проектов (`*.png`) + стрелка (`direct.svg`) |

## 🏗️ Вёрстка

- **Контейнер**: `.slider-wrapper` (1440×783px) центрируется через `margin: 0 auto`. Внутри `.container` (1171×552px) с `position: relative`.
- **Позиционирование**: Все внутренние блоки привязаны абсолютно к `.container` (пиксельная точность по Figma). Адаптивность не требуется.
- **Навигация сверху**: 3 ссылки (`data-index="0|1|2"`) + 3 независимых блока подчёркивания (каждый со своей `width`).
- **Контент**: Левый текстовый блок (заголовок, описание, сетка 2×2) + правый блок с фоновым изображением.
- **Управление**: Стрелки (`<img src="direct.svg">`), контейнер с точками. Левая стрелка зеркалится через `transform: scaleX(-1)`.

## ⚙️ Логика JS

- **Данные**: Массив `slides` (JSON-структура). Ключи: `id`, `img`, `scale`, `city`, `area`, `time`, `request`.
- **Индексация**: `0` = левый, `1` = средний, `2` = правый. Строго синхронизирована с HTML/CSS.
- **Кольцевое переключение**: `currentIndex = ((index % total) + total) % total`.
- **Обновление при клике**:
  - Картинка: `backgroundImage` + `transform: scale(data.scale)` (динамически из JSON).
  - Текст: `innerHTML` для `city` (поддержка `<br>`), `textContent` для остальных полей.
  - UI: переключение классов `.active`, `.visible` у ссылок, точек и подчёркиваний.

## 🔑 Ключевые решения

| Проблема                    | Решение                                                                              |
| --------------------------- | ------------------------------------------------------------------------------------ |
| Мерцание при смене фона     | `transition: none` у `.slider-image`. Смена мгновенная, без интерполяции.            |
| Прозрачные поля у картинок  | Индивидуальный `scale` в JSON. Задаётся inline через JS.                             |
| Рассинхрон навигации        | Единый `currentIndex` управляет ссылками, точками, подчёркиваниями и контентом.      |
| Разная ширина подчёркиваний | 3 независимых DOM-элемента, переключаются через `visible`.                           |
| Перенос строки в `city`     | `<br>` в JSON + `innerHTML` в JS (остальные поля на `textContent` для безопасности). |

## 📥 Интеграция

1. Перенести файлы и папку `img/` в проект.
2. Подключить Google Fonts: `Yeseva One`, `Nunito Sans`.
3. Вставить HTML-разметку слайдера в нужное место страницы.
4. Проверить пути к `script.js` и `img/`.
5. При необходимости сменить стартовый слайд: в `DOMContentLoaded` вызвать `setSlide(N)`.

✅ **Готово к использованию.** Автономен, не требует зависимостей, логика замкнута в `script.js`.
